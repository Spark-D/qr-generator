# WIFI QR 코드 카드 생성기 PRD

## 프로젝트 개요
매장에서 고객들이 쉽게 WIFI에 접속할 수 있도록 QR 코드 카드를 생성하고 출력할 수 있는 웹 서비스입니다.

## 핵심 기능 요구사항

### 1. QR 코드 카드 생성
- **브랜드 이름**: 매장명 또는 브랜드명 입력
- **네트워크 이름**: WIFI SSID 입력
- **비밀번호**: WIFI 비밀번호 입력
- **카드 배경 색상**: 사용자 정의 색상 선택 가능

### 2. 카드 디자인 사양
- **카드 크기**: 256px 고정
- **가로세로 비율**: 3:4 (192px × 256px)
- **디자인 스타일**: 미니멀한 디자인
- **레이아웃 구성**:
  - 상단: "WIFI 접속" 텍스트
  - 중앙: QR 코드
  - 하단: 브랜드 이름
  - 하단 우측: "by SPARK" 레이블 (작은 크기)

### 3. 사용자 인터페이스
- **입력 폼**: 브랜드명, SSID, 비밀번호, 배경색 입력
- **실시간 미리보기**: 입력값 변경 시 즉시 카드 미리보기 업데이트
- **색상 선택기**: 배경색 커스터마이징

### 4. 출력 기능
- **이미지 다운로드**: 생성된 카드를 PNG/JPEG 이미지로 다운로드
- **고해상도**: 인쇄용 고품질 이미지 생성
- **파일명**: 브랜드명_QR카드 형식으로 자동 생성

## 기술적 요구사항

### 프론트엔드
- **QR 코드 생성**: qrcode 라이브러리 사용
- **이미지 다운로드**: HTML5 Canvas API 활용
- **폼 관리**: react-hook-form 라이브러리 사용
- **반응형 디자인**: 모바일/데스크톱 대응
- **실시간 미리보기**: 입력값 변경 시 즉시 반영

### 사용자 경험
- **직관적 인터페이스**: 간단한 폼 입력으로 카드 생성
- **즉시 미리보기**: 입력과 동시에 결과 확인
- **원클릭 다운로드**: 생성된 카드 즉시 다운로드
- **한국어 지원**: 모든 텍스트 한국어 기본 설정

## 성능 요구사항
- **빠른 렌더링**: QR 코드 생성 및 미리보기 즉시 표시
- **최적화된 이미지**: 다운로드 시 적절한 파일 크기
- **브라우저 호환성**: 최신 브라우저 지원

## 보안 고려사항
- **클라이언트 사이드**: 모든 처리는 브라우저에서 수행
- **데이터 저장**: 사용자 입력 데이터 서버 저장 안함
- **개인정보**: WIFI 정보는 로컬에서만 처리

## 향후 확장 가능성
- **템플릿 선택**: 다양한 카드 디자인 템플릿
- **로고 업로드**: 브랜드 로고 추가 기능
- **배치 생성**: 여러 매장 정보 일괄 처리
- **인쇄 가이드**: 인쇄 시 주의사항 안내
